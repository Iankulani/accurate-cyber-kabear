# accurate-cyber-kabear
Accurate Cyber KaBear is a lightweight but powerful network utility designed for port scanning, device reachability checks (ping), IP/service monitoring, and real-time alerting. 
It is built to run efficiently on anything from enterprise servers to Raspberry Pi, making it ideal for cybersecurity professionals, sysadmins, and hobbyists.

## How to clone the repo

```bash

git clone https://github.com/Iankulani/accurate-cyber-kabear.git

cd accurate-cyber-kabear
```

## How to run
```bash

python3 accurate-cyber-kabear.py

```
